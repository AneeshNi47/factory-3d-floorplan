# 3D Factory Floor Plan — Coimbatore Manufacturing Unit

Interactive 3D floor plan for a multi-product manufacturing facility at SIPCOT Coimbatore, Tamil Nadu.

## Features
- 🏭 **13 Production Lines** with ~35 optimized machines
- 🔧 **Individual 3D machine models** — CNC mills, laser cutters, roll formers, presses, welding stations, etc.
- 🖱️ **Interactive** — Drag to rotate, scroll to zoom, click any zone/machine for details
- 🎬 **Guided Tour** — Automated flythrough of all production zones
- 📊 **Cost annotations** — ₹21 Cr optimized capex breakdown per zone
